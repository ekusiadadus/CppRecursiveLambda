## Summary

## How to Build

This project use [CMake](https://cmake.org/ "CMake").
So you can build appropriately.

Following operation is one example to build.

```shell
$ mkdir build
$ cd build
$ cmake .. -DCMAKE_BUILD_TYPE=Release
$ make
```
